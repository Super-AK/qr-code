# Changelog

## [3.0.0] - 2026-07-14

### Modernisierung
- **Vite** als Build-System (Hot-Reload, optimierte Bundles)
- **TypeScript** für Typsicherheit
- **Modulare Architektur** (ES-Module statt monolithischer Datei)

### Neue QR-Typen
- WhatsApp (Direktnachricht)
- Kalender (Event/Termin)
- Geolokalisierung (Standort)
- Kryptowährung (Bitcoin, Ethereum, Litecoin)
- Soziale Medien (Instagram, Facebook, etc.)

### UI/UX
- Tab-basierte QR-Typ-Auswahl (statt Dropdown)
- Design-Optionen (QR-Form: Quadrat, Abgerundet, Kreis)
- Professionelleres Layout

### Technologien
- Vite 6.0
- TypeScript 5.5
- Three.js 0.170 (aktuelle Version)
- Bootstrap 5.3.2
